# Phonebook API

Api for a phonebook project.

Link demo: https://phonebook-api-1ul3.onrender.com/api/persons/

## Project Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development mode (with nodemon):
   ```bash
   npm run dev
   ```

3. Run in production mode:
   ```bash
   npm start
   ```

## Technologies
- Node.js
- Express
- Nodemon (dev dependency)
